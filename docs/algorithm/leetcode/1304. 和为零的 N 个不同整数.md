## 题目

给你一个整数 n，请你返回 任意 一个由 n 个 各不相同 的整数组成的数组，并且这 n 个数相加和为 0 。



示例 1：

    输入：n = 5
    输出：[-7,-1,1,3,4]
    解释：这些数组也是正确的 [-5,-1,1,2,3]，[-3,-1,2,-2,4]。
示例 2：

    输入：n = 3
    输出：[-1,0,1]
示例 3：

    输入：n = 1
    输出：[0]


提示：

* 1 <= n <= 1000

## 思路

二分法

## 解法
```java
class Solution {
    public int[] sumZero(int n) {
        int[] num = new int[n];
        int hight=n;
        int low = 1;
        if(n%2==0){
           while(hight != low-1){
               num[low-1] = low;
               num[hight-1] = -low;
               low++;
               hight--;
           }
        }else{
            num[n/2] = 0;
            while(hight != low){
                num[low-1] = low;
               num[hight-1] = -low;
               low++;
               hight--;
            }
        }
        return num;

    }
}

```

## 总结

- 分析出几种情况，然后分别对各个情况实现
