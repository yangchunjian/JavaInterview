## 题目

给你一个整数 n 。如果 n 恰好有三个正除数 ，返回 true ；否则，返回 false 。

如果存在整数 k ，满足 n = k * m ，那么整数 m 就是 n 的一个 除数 。



示例 1：

    输入：n = 2
    输出：false
    解释：2 只有两个除数：1 和 2 。
示例 2：

    输入：n = 4
    输出：true
    解释：4 有三个除数：1、2 和 4 。


提示：

* 1 <= n <= 10<sup>4</sup>

## 思路

for

## 解法
```java
class Solution {
    public boolean isThree(int n) {
        for(int i = 2;i*i<=n;i++){
            if(i*i==n) return true;
            if(n%i==0) return false;
        }
        return false;
    }
}

```

## 总结

- 分析出几种情况，然后分别对各个情况实现
