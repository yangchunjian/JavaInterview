## 题目
给你一个二进制数组 nums ，你需要从中删掉一个元素。

请你在删掉元素的结果数组中，返回最长的且只包含 1 的非空子数组的长度。

如果不存在这样的子数组，请返回 0 。



提示 1：

    输入：nums = [1,1,0,1]
    输出：3
    解释：删掉位置 2 的数后，[1,1,1] 包含 3 个 1 。
示例 2：

    输入：nums = [0,1,1,1,0,1,1,0,1]
    输出：5
    解释：删掉位置 4 的数字后，[0,1,1,1,1,1,0,1] 的最长全 1 子数组为 [1,1,1,1,1] 。
示例 3：

    输入：nums = [1,1,1]
    输出：2
    解释：你必须要删除一个元素。


提示：

* 1 <= nums.length <= 105
* nums[i] 要么是 0 要么是 1 。



## 思路

解决不定长滑动数组的一个常见模板



## 解法
```java
class Solution {
    public int longestSubarray(int[] nums) {
        // 其实只需要统计0的数量
        int maxResult = 0;
        int left = 0;
        int zeroCount = 0;

        for (int right = 0; right < nums.length; right++) {
            if (nums[right] == 0) {
                zeroCount++;
            }

            while (zeroCount > 1) {
                if (nums[left] == 0) {
                    zeroCount--;
                }
                left++;
            }

            maxResult = Math.max(maxResult, right - left);
        }

        return maxResult;
    }
}

```
