# 怎么配置

  1. 下载mac goland软件
  2. 配置gopath、goroot
  3. which go后，在go的目录下执行get github.com/gogo/protobuf/protoc-gen-gogo安装文件
  4. 打开goland软件，在项目的根目录下执行make p dep
  5. 即可command+鼠标，进入方法内部查看
