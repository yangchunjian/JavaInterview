# еөҢеҘ—зұ»зҡ„и®ҝй—®жҺ§еҲ¶и§„еҲҷ

Java 11д№ӢеүҚпјҢеҶ…йғЁеөҢеҘ—зұ»и®ҝй—®еӨ–йғЁзұ»зҡ„з§ҒжңүеұһжҖ§е’Ңж–№жі•жҳҜеҸҜиЎҢзҡ„пјҡ

JVM и®ҝй—®и§„еҲҷдёҚе…Ғи®ёеөҢеҘ—зұ»д№Ӣй—ҙиҝӣиЎҢз§Ғжңүи®ҝй—®гҖӮжҲ‘д»¬иғҪйҖҡиҝҮеёёи§„ж–№ејҸеҸҜд»Ҙи®ҝй—®жҳҜеӣ дёә JVM еңЁзј–иҜ‘ж—¶дёәжҲ‘д»¬йҡҗејҸең°еҲӣе»әдәҶжЎҘжҺҘж–№жі•гҖӮJava 11 дёӯеј•е…ҘдәҶдёӨдёӘж–°зҡ„еұһжҖ§пјҡдёҖдёӘеҸ«еҒҡ NestMembers зҡ„еұһжҖ§пјҢз”ЁдәҺж ҮиҜҶе…¶е®ғе·ІзҹҘзҡ„йқҷжҖҒ nest жҲҗе‘ҳпјӣеҸҰеӨ–дёҖдёӘжҳҜжҜҸдёӘ nest жҲҗе‘ҳйғҪеҢ…еҗ«зҡ„ NestHost еұһжҖ§пјҢз”ЁдәҺж ҮиҜҶеҮәе®ғзҡ„ nest е®ҝдё»зұ»гҖӮеңЁзј–иҜ‘жңҹе°ұжҳ е°„дәҶеҸҢж–№зҡ„еҜ„е®ҝе…ізі»пјҢдёҚеҶҚйңҖиҰҒжЎҘжҺҘдәҶгҖӮ
