# дёҖгҖҒRedisжЁЎејҸ

Redisжңүдёүз§ҚжЁЎејҸпјҡеҲҶеҲ«жҳҜдё»д»ҺеҗҢжӯҘ/еӨҚеҲ¶гҖҒе“Ёе…өжЁЎејҸгҖҒCluster

  • дё»д»ҺеӨҚеҲ¶пјҡдё»д»ҺеӨҚеҲ¶жҳҜй«ҳеҸҜз”ЁRedisзҡ„еҹәзЎҖпјҢе“Ёе…өе’ҢзҫӨйӣҶйғҪжҳҜеңЁдё»д»ҺеӨҚеҲ¶еҹәзЎҖдёҠе®һзҺ°й«ҳеҸҜз”Ёзҡ„гҖӮдё»д»ҺеӨҚеҲ¶дё»иҰҒе®һзҺ°дәҶж•°жҚ®зҡ„еӨҡжңәеӨҮд»ҪпјҢд»ҘеҸҠеҜ№дәҺиҜ»ж“ҚдҪңзҡ„иҙҹиҪҪеқҮиЎЎе’Ңз®ҖеҚ•ж•…йҡңжҒўеӨҚгҖӮ
  • зјәйҷ·пјҡж•…йҡңжҒўеӨҚж— жі•иҮӘеҠЁеҢ–пјҢеҶҷж“ҚдҪңж— жі•иҙҹиҪҪеқҮиЎЎпјҢеӯҳеӮЁиғҪеҠӣеҸ—еҲ°еҚ•жңәзҡ„йҷҗеҲ¶гҖӮ
  • е“Ёе…өпјҡеңЁдё»д»ҺеӨҚеҲ¶зҡ„еҹәзЎҖдёҠпјҢе“Ёе…өе®һзҺ°дәҶиҮӘеҠЁеҢ–зҡ„ж•…йҡңжҒўеӨҚгҖӮ
  • зјәйҷ·пјҡеҶҷж“ҚдҪңж— жі•иҙҹиҪҪеқҮиЎЎпјҢеӯҳеӮЁиғҪеҠӣеҸ—еҲ°еҚ•жңәзҡ„йҷҗеҲ¶пјҢе“Ёе…өж— жі•еҜ№д»ҺиҠӮзӮ№иҝӣиЎҢиҮӘеҠЁж•…йҡңиҪ¬з§»пјӣеңЁиҜ»еҶҷеҲҶзҰ»еңәжҷҜдёӢпјҢд»ҺиҠӮзӮ№ж•…йҡңдјҡеҜјиҮҙиҜ»жңҚеҠЎдёҚеҸҜз”ЁпјҢйңҖиҰҒеҜ№д»ҺиҠӮзӮ№еҒҡйўқеӨ–зҡ„зӣ‘жҺ§гҖҒеҲҮжҚўж“ҚдҪңгҖӮ
  • йӣҶзҫӨпјҡйҖҡиҝҮйӣҶзҫӨпјҢRedisи§ЈеҶідәҶеҶҷж“ҚдҪңж— жі•иҙҹиҪҪеқҮиЎЎпјҢд»ҘеҸҠеӯҳеӮЁиғҪеҠӣеҸ—еҲ°еҚ•жңәйҷҗеҲ¶зҡ„й—®йўҳпјҢе®һзҺ°дәҶиҫғдёәе®Ңе–„зҡ„й«ҳеҸҜз”Ёж–№жЎҲгҖӮ

# дәҢгҖҒRedisдё»д»ҺеӨҚеҲ¶

2.1 дё»д»ҺеӨҚеҲ¶жҰӮиҝ°

  • дё»д»ҺеӨҚеҲ¶пјҢжҳҜжҢҮе°ҶдёҖеҸ° Redis жңҚеҠЎеҷЁзҡ„ж•°жҚ®пјҢеӨҚеҲ¶еҲ°е…¶д»–зҡ„ Redis жңҚеҠЎеҷЁгҖӮеүҚиҖ…з§°дёәдё»иҠӮзӮ№(Master)пјҢеҗҺиҖ…з§°дёәд»ҺиҠӮзӮ№(Slave)пјӣж•°жҚ®зҡ„еӨҚеҲ¶жҳҜеҚ•еҗ‘зҡ„пјҢеҸӘиғҪз”ұдё»иҠӮзӮ№еҲ°д»ҺиҠӮзӮ№гҖӮ
  • й»ҳи®Өжғ…еҶөдёӢпјҢжҜҸеҸ° Redis жңҚеҠЎеҷЁйғҪжҳҜдё»иҠӮзӮ№пјӣдё”дёҖдёӘдё»иҠӮзӮ№еҸҜд»ҘжңүеӨҡдёӘд»ҺиҠӮзӮ№ (жҲ–жІЎжңүд»ҺиҠӮзӮ№)пјҢдҪҶдёҖдёӘд»ҺиҠӮзӮ№еҸӘиғҪжңүдёҖдёӘдё»иҠӮзӮ№гҖӮ

2.2 дё»д»ҺеӨҚеҲ¶

  • ж•°жҚ®еҶ—дҪҷпјҡдё»д»ҺеӨҚеҲ¶е®һзҺ°дәҶж•°жҚ®зҡ„зғӯеӨҮд»ҪпјҢжҳҜжҢҒд№…еҢ–д№ӢеӨ–зҡ„дёҖз§Қж•°жҚ®еҶ—дҪҷж–№ејҸгҖӮ
  • ж•…йҡңжҒўеӨҚпјҡеҪ“дё»иҠӮзӮ№еҮәзҺ°й—®йўҳж—¶пјҢеҸҜд»Ҙз”ұд»ҺиҠӮзӮ№жҸҗдҫӣжңҚеҠЎпјҢе®һзҺ°еҝ«йҖҹзҡ„ж•…йҡңжҒўеӨҚпјӣе®һйҷ…дёҠжҳҜдёҖз§ҚжңҚеҠЎзҡ„еҶ—дҪҷгҖӮ
  • иҙҹиҪҪеқҮиЎЎпјҡеңЁдё»д»ҺеӨҚеҲ¶зҡ„еҹәзЎҖдёҠпјҢй…ҚеҗҲиҜ»еҶҷеҲҶзҰ»пјҢеҸҜд»Ҙз”ұдё»иҠӮзӮ№жҸҗдҫӣеҶҷжңҚеҠЎпјҢз”ұд»ҺиҠӮзӮ№жҸҗдҫӣиҜ»жңҚеҠЎ (еҚіеҶҷ Redis ж•°жҚ®ж—¶еә”з”ЁиҝһжҺҘдё»иҠӮзӮ№пјҢиҜ» Redis ж•°жҚ®ж—¶еә”з”ЁиҝһжҺҘд»ҺиҠӮзӮ№)пјҢеҲҶжӢ…жңҚеҠЎеҷЁиҙҹиҪҪпјӣе°Өе…¶жҳҜеңЁеҶҷе°‘иҜ»еӨҡзҡ„еңәжҷҜдёӢпјҢйҖҡиҝҮеӨҡдёӘд»ҺиҠӮзӮ№еҲҶжӢ…иҜ»иҙҹиҪҪпјҢеҸҜд»ҘеӨ§еӨ§жҸҗй«ҳRedisжңҚеҠЎеҷЁзҡ„е№¶еҸ‘йҮҸгҖӮ
  • й«ҳеҸҜз”ЁеҹәзҹіпјҡйҷӨдәҶдёҠиҝ°дҪңз”Ёд»ҘеӨ–пјҢдё»д»ҺеӨҚеҲ¶иҝҳжҳҜе“Ёе…өе’ҢйӣҶзҫӨиғҪеӨҹе®һж–Ҫзҡ„еҹәзЎҖпјҢеӣ жӯӨиҜҙдё»д»ҺеӨҚеҲ¶жҳҜRedisй«ҳеҸҜз”Ёзҡ„еҹәзЎҖгҖӮ

2.3 Redisдё»д»ҺеӨҚеҲ¶жөҒзЁӢ

  • иӢҘеҗҜеҠЁдёҖдёӘSlaveжңәеҷЁиҝӣзЁӢпјҢеҲҷе®ғдјҡеҗ‘MasterжңәеҷЁеҸ‘йҖҒдёҖдёӘвҖңsync command" е‘Ҫд»ӨпјҢиҜ·жұӮеҗҢжӯҘиҝһжҺҘгҖӮ
  • ж— и®әжҳҜз¬¬дёҖж¬ЎиҝһжҺҘиҝҳжҳҜйҮҚж–°иҝһжҺҘпјҢMasterжңәеҷЁ йғҪдјҡеҗҜеҠЁдёҖдёӘеҗҺеҸ°иҝӣзЁӢпјҢе°Ҷж•°жҚ®еҝ«з…§дҝқеӯҳеҲ°ж•°жҚ®ж–Үд»¶дёӯ(жү§иЎҢrdbж“ҚдҪң) пјҢеҗҢж—¶ Master иҝҳдјҡи®°еҪ•дҝ®ж”№ж•°жҚ®зҡ„жүҖжңүе‘Ҫд»Өе№¶зј“еӯҳеңЁж•°жҚ®ж–Үд»¶дёӯгҖӮ
  • еҗҺеҸ°иҝӣзЁӢе®ҢжҲҗзј“еӯҳж“ҚдҪңд№ӢеҗҺпјҢMaster жңәеҷЁе°ұдјҡеҗ‘ Slave жңәеҷЁеҸ‘йҖҒж•°жҚ®ж–Үд»¶пјҢSlave з«ҜжңәеҷЁе°Ҷж•°жҚ®ж–Үд»¶дҝқеӯҳеҲ°зЎ¬зӣҳдёҠпјҢз„¶еҗҺе°Ҷе…¶еҠ иҪҪеҲ°еҶ…еӯҳдёӯпјҢжҺҘзқҖ Master жңәеҷЁе°ұдјҡе°Ҷдҝ®ж”№ж•°жҚ®зҡ„жүҖжңүж“ҚдҪңдёҖе№¶еҸ‘йҖҒз»ҷ Slave з«ҜжңәеҷЁгҖӮиӢҘ Slave еҮәзҺ°ж•…йҡңеҜјиҮҙе®•жңәпјҢеҲҷжҒўеӨҚжӯЈеёёеҗҺдјҡиҮӘеҠЁйҮҚж–°иҝһжҺҘгҖӮ
  • MasterжңәеҷЁж”¶еҲ° Slave з«ҜжңәеҷЁзҡ„иҝһжҺҘеҗҺпјҢе°Ҷе…¶е®Ңж•ҙзҡ„ж•°жҚ®ж–Үд»¶еҸ‘йҖҒз»ҷ Slave з«ҜжңәеҷЁпјҢеҰӮжһң Mater еҗҢж—¶ж”¶еҲ°еӨҡдёӘ Slave еҸ‘жқҘзҡ„еҗҢжӯҘиҜ·жұӮпјҢеҲҷ Master дјҡеңЁеҗҺеҸ°еҗҜеҠЁдёҖдёӘиҝӣзЁӢд»Ҙдҝқеӯҳж•°жҚ®ж–Үд»¶пјҢз„¶еҗҺе°Ҷе…¶еҸ‘йҖҒз»ҷжүҖжңүзҡ„ Slave з«ҜжңәеҷЁпјҢзЎ®дҝқжүҖжңүзҡ„ Slave з«ҜжңәеҷЁйғҪжӯЈеёёгҖӮ

# дә”гҖҒRedisйӣҶзҫӨжЁЎејҸ

  • йӣҶзҫӨпјҢеҚі Redis ClusterпјҢ жҳҜRedis 3. 0ејҖе§Ӣеј•е…Ҙзҡ„еҲҶеёғејҸеӯҳеӮЁж–№жЎҲгҖӮ
  • йӣҶзҫӨз”ұеӨҡдёӘиҠӮзӮ№(Node) з»„жҲҗпјҢRedis зҡ„ж•°жҚ®еҲҶеёғеңЁиҝҷдәӣиҠӮзӮ№дёӯгҖӮ
  • йӣҶзҫӨдёӯзҡ„иҠӮзӮ№еҲҶдёәдё»иҠӮзӮ№е’Ңд»ҺиҠӮзӮ№пјӣеҸӘжңүдё»иҠӮзӮ№иҙҹиҙЈиҜ»еҶҷиҜ·жұӮе’ҢйӣҶзҫӨдҝЎжҒҜзҡ„з»ҙжҠӨпјӣд»ҺиҠӮзӮ№еҸӘиҝӣиЎҢдё»иҠӮзӮ№ж•°жҚ®е’ҢзҠ¶жҖҒдҝЎжҒҜзҡ„еӨҚеҲ¶гҖӮ

5.1 йӣҶзҫӨзҡ„дҪңз”Ё

1.ж•°жҚ®еҲҶеҢәпјҡж•°жҚ®еҲҶеҢә(жҲ–з§°ж•°жҚ®еҲҶзүҮ) жҳҜйӣҶзҫӨжңҖж ёеҝғзҡ„еҠҹиғҪгҖӮ

  • йӣҶзҫӨе°Ҷж•°жҚ®еҲҶж•ЈеҲ°еӨҡдёӘиҠӮзӮ№пјҢдёҖж–№йқўзӘҒз ҙдәҶ Redis еҚ•жңәеҶ…еӯҳеӨ§е°Ҹзҡ„йҷҗеҲ¶пјҢеӯҳеӮЁе®№йҮҸеӨ§еӨ§еўһеҠ пјӣеҸҰдёҖж–№йқўжҜҸдёӘдё»иҠӮзӮ№йғҪеҸҜд»ҘеҜ№еӨ–жҸҗдҫӣиҜ»жңҚеҠЎе’ҢеҶҷжңҚеҠЎпјҢеӨ§еӨ§жҸҗй«ҳдәҶйӣҶзҫӨзҡ„е“Қеә”иғҪеҠӣгҖӮ
  • Redis еҚ•жңәеҶ…еӯҳеӨ§е°ҸеҸ—йҷҗй—®йўҳпјҢеңЁд»Ӣз»ҚжҢҒд№…еҢ–е’Ңдё»д»ҺеӨҚеҲ¶ж—¶йғҪжңүжҸҗеҸҠпјӣдҫӢеҰӮпјҢеҰӮжһңеҚ•жңәеҶ…еӯҳеӨӘеӨ§пјҢbgsave е’Ң bgrewriteaofзҡ„ fork ж“ҚдҪңеҸҜиғҪеҜјиҮҙдё»иҝӣзЁӢйҳ»еЎһпјҢдё»д»ҺзҺҜеўғдёӢдё»жңәеҲҮжҚўж—¶еҸҜиғҪеҜјиҮҙд»ҺиҠӮзӮ№й•ҝж—¶й—ҙж— жі•жҸҗдҫӣжңҚеҠЎпјҢе…ЁйҮҸеӨҚеҲ¶йҳ¶ж®өдё»иҠӮзӮ№зҡ„еӨҚеҲ¶зј“еҶІеҢәеҸҜиғҪжәўеҮәгҖӮ

2.й«ҳеҸҜз”ЁпјҡйӣҶзҫӨж”ҜжҢҒдё»д»ҺеӨҚеҲ¶е’Ңдё»иҠӮзӮ№зҡ„иҮӘеҠЁж•…йҡңиҪ¬з§»(дёҺе“Ёе…өзұ»дјј) пјӣеҪ“д»»дёҖиҠӮзӮ№еҸ‘з”ҹж•…йҡңж—¶пјҢйӣҶзҫӨд»Қз„¶еҸҜд»ҘеҜ№еӨ–жҸҗдҫӣжңҚеҠЎгҖӮ

5.2 йӣҶзҫӨжЁЎејҸзҡ„ж•°жҚ®еҲҶзүҮ

  • RedisйӣҶзҫӨеј•е…ҘдәҶе“ҲеёҢж§Ҫзҡ„жҰӮеҝө
  • RedisйӣҶзҫӨжңү 16384 дёӘе“ҲеёҢж§Ҫ( зј–еҸ·0-16383)
  • йӣҶзҫӨзҡ„жҜҸдёӘиҠӮзӮ№иҙҹиҙЈдёҖйғЁеҲҶе“ҲеёҢж§Ҫ
  • жҜҸдёӘKey йҖҡиҝҮ CRC16 ж ЎйӘҢеҗҺеҜ№16384еҸ–дҪҷжқҘеҶіе®ҡж”ҫзҪ®е“ӘдёӘе“ҲеёҢж§ҪпјҢйҖҡиҝҮиҝҷдёӘеҖјпјҢеҺ»жүҫеҲ°еҜ№еә”зҡ„жҸ’ж§ҪжүҖеҜ№еә”зҡ„иҠӮзӮ№пјҢз„¶еҗҺзӣҙжҺҘиҮӘеҠЁи·іиҪ¬еҲ°иҝҷдёӘеҜ№еә”зҡ„иҠӮзӮ№дёҠиҝӣиЎҢеӯҳеҸ–ж“ҚдҪң

вҖӢвҖӢвҖӢд»Ҙ3дёӘиҠӮзӮ№з»„жҲҗзҡ„йӣҶзҫӨдёәдҫӢвҖӢвҖӢвҖӢ

5.3 йӣҶзҫӨжЁЎејҸзҡ„дё»д»ҺеӨҚеҲ¶жЁЎеһӢ

  • йӣҶзҫӨдёӯе…·жңүAгҖҒBгҖҒCдёүдёӘиҠӮзӮ№пјҢеҰӮжһңиҠӮзӮ№BеӨұиҙҘдәҶпјҢж•ҙдёӘйӣҶзҫӨе°ұдјҡеӣ зјәе°‘5461-10922иҝҷдёӘиҢғеӣҙзҡ„ж§ҪиҖҢдёҚеҸҜд»Ҙз”ЁгҖӮ
  • дёәжҜҸдёӘиҠӮзӮ№ж·»еҠ дёҖдёӘд»ҺиҠӮзӮ№A1гҖҒB1гҖҒC1ж•ҙдёӘйӣҶзҫӨдҫҝжңүдёүдёӘMasterиҠӮзӮ№е’ҢдёүдёӘslaveиҠӮзӮ№з»„жҲҗпјҢеңЁиҠӮзӮ№BеӨұиҙҘеҗҺпјҢйӣҶзҫӨйҖүдёҫB1дҪҚдёәдё»иҠӮзӮ№з»§з»ӯжңҚеҠЎгҖӮеҪ“Bе’ҢB1йғҪеӨұиҙҘеҗҺпјҢйӣҶзҫӨе°ҶдёҚеҸҜз”ЁгҖӮ
